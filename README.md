# MyCopilot AI对话应用Demo

这里是**MyCopilot** AI对话应用Demo的代码仓库，项目尚处于准备阶段，本文档会随着项目进展持续更新。

##  项目简介

MyCopilot 是一个现代化的 AI 对话应用 Demo，采用 React + TypeScript 构建，提供流畅的对话体验和护眼主题配色方案。项目旨在展示如何构建一个功能完整、性能优秀的 AI 对话界面。


##  技术栈

### 核心框架
- **开发框架**：React 19 + TypeScript
- **样式框架**：TailwindCSS 4
- **状态管理**：Zustand
- **打包工具**：Vite (rolldown)
- **依赖管理**：pnpm
- **开发工具**：Cursor

### 主要依赖
- `react` / `react-dom` - React 核心库
- `tailwindcss` - 原子化 CSS 框架
- `@tailwindcss/vite` - TailwindCSS Vite 插件
- `@vitejs/plugin-react-swc` - React SWC 插件，提供快速编译

### 开发工具
- `typescript` - TypeScript 支持
- `eslint` - 代码检查工具
- `typescript-eslint` - TypeScript ESLint 支持


**注意**：本项目目前处于准备阶段，功能正在逐步完善中。
